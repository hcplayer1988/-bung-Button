Aufgabe 1

Schreibe eine Funktion, die eine Zahl als Note (zwischen 1 und 6) entgegennimmt und eine entsprechende Nachricht in der Konsole ausgibt, ob die Note „bestanden“ (Note 1-4) oder „nicht bestanden“ (Note 5 und 6) ist.

Aufgabe 2

Schreibe eine Funktion, die prüft, ob eine Zahl positiv, negativ oder gleich 0 ist, und eine entsprechende Nachricht in der Konsole zurückgibt.

Aufgabe 3

Schreibe eine Funktion, die zwei Zahlen als Parameter bekommt und die größere der beiden Zahlen in der Konsole zurückgibt.

Zusatzaufgabe

Schreibe eine Funktion, welche einen String als Parameter erhält und überprüft, ob dieser mehr als 5 Zeichen hat. Sende eine entsprechende Nachricht zurück.
